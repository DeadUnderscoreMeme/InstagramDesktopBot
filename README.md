# InstagramDesktopBot

This is a desktop client that allows you to view your Instagram stats, posts and more. <br>

>What you can do with this client right now is : <br>

The basic functions of this bot are : <br>
1. With Users
    1. Find by username
    2. Follow them
2. With Feed
    1. Access Posts
    2. Access Profile Pictures
    3. Access Post Image
    4. Like Button
    5. Comment Space
    6. Share Button
3. With Stories
    1. View all stories of the user
    2. View Stories of a specific user after visting their profile
